# Manual Tests

このディレクトリには、手動で実行するテスト手順が含まれます。

## テストドキュメント

- `test_procedure.md`: 詳細なマニュアルテスト手順
- `daw_compatibility.md`: DAW互換性チェックリスト
- `audio_quality.md`: 音質チェックリスト

## 必要な環境

- DAW（Reaper推奨、他にAbleton Live、FL Studio、Cubaseなど）
- テスト用オーディオファイル
  - ピンクノイズ（ステレオ）
  - サイン波（1kHz、ステレオ）
  - 楽曲サンプル（ステレオ）
- ヘッドフォンまたはモニタースピーカー

## テスト手順の概要

1. プラグインを読み込む
2. デフォルト状態で音が変わらないことを確認（バイパステスト）
3. 各パラメータを個別に変更して動作確認
4. 複数パラメータの組み合わせテスト
5. オートメーションテスト
6. プロジェクト保存・読み込みテスト
7. CPU使用率チェック

詳細は `test_procedure.md` を参照してください。
