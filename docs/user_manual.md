## 概要

SimplePanner は、ステレオ入力信号の各チャンネル（L/R）を独立して制御できる VST3 オーディオプラグインです。パンニング、ゲイン調整、遅延を組み合わせることで、ステレオイメージの精密なコントロールとハース効果による定位調整を実現します。

### 主な機能

- ✨ **独立した L/R チャンネル制御**: 左右の入力を完全に独立して処理
- 🎚️ **精密なパンニング**: 等パワーパンニングによる自然な定位
- 🔊 **柔軟なゲイン調整**: -60dB 〜 +6dB の広範囲なゲイン制御
- ⏱️ **ハース効果対応**: 0〜100ms の遅延で定位感を創出
- 🔗 **L/R ゲイン連動**: ステレオバランスを保ったまま音量調整

## インストール

### 必要環境

- **DAW**: VST3 対応の Digital Audio Workstation
  - Reaper、Cubase、Studio One、FL Studio、Ardour など
- **OS**:
  - Windows 10/11 (64-bit)
  - macOS 10.12 以降
  - Linux (Ubuntu 18.04 以降推奨)
- **CPU**: Intel/AMD 64-bit プロセッサ
- **RAM**: 最小 4GB

### インストール手順

1. **プラグインファイルの入手**
   - リリースページから `SimplePanner.vst3` をダウンロード

2. **VST3 フォルダにコピー**

   **Windows**:
   ```
   C:\Program Files\Common Files\VST3\
   ```

   **macOS**:
   ```
   /Library/Audio/Plug-Ins/VST3/
   または
   ~/Library/Audio/Plug-Ins/VST3/
   ```

   **Linux**:
   ```
   ~/.vst3/
   ```

3. **DAW でプラグインをスキャン**
   - DAW を起動し、プラグインスキャンを実行
   - SimplePanner が FX プラグインリストに表示されることを確認

## クイックスタート

### 基本的な使い方

1. **プラグインの挿入**
   - DAW でステレオトラックを作成
   - トラックに SimplePanner を挿入

2. **デフォルト状態の確認**
   - デフォルトでは入力信号がそのまま出力されます（透過状態）
   - Left Pan = -100 (Full Left)
   - Right Pan = +100 (Full Right)
   - すべての Gain = 0dB

3. **パラメータ調整**
   - GUI でスライダーやノブを操作
   - リアルタイムで値が表示されます

### 5分間チュートリアル

#### 例1: ステレオイメージを広げる

1. Left Pan を -100 のまま
2. Right Pan を +100 のまま
3. Left Delay を 10ms に設定
4. Right Delay を 0ms のまま
5. → 左チャンネルがわずかに遅延し、ステレオイメージが広がります

#### 例2: L/R をスワップ

1. Left Pan を +100 に変更（右端までドラッグ）
2. Right Pan を -100 に変更（左端までドラッグ）
3. → 左右のチャンネルが入れ替わります

#### 例3: モノラル化

1. Left Pan を 0 に設定（中央）
2. Right Pan を 0 に設定（中央）
3. → 両チャンネルがセンターに定位します

## GUI リファレンス

### GUI 概要

SimplePanner は 600x400 ピクセルのコンパクトなインターフェイスを提供します。

```
┌─────────────────────────────────────────────────────────┐
│                    SimplePanner                          │
├──────────────────────┬──────────────────────────────────┤
│   LEFT CHANNEL       │      RIGHT CHANNEL               │
│                      │                                  │
│  Pan:  L ━━●━━━ R   │   Pan:  L ━━━━●━ R             │
│        "L100"        │         "R50"                    │
│                      │                                  │
│  Gain   Delay        │   Gain    Delay                 │
│   ◯      ◯          │    ◯       ◯                   │
│  0.0dB  10ms         │  -3dB    0.0ms                  │
│                      │                                  │
├──────────────────────┴──────────────────────────────────┤
│                    MASTER                                │
│                                                          │
│     Master Gain        [  LINKED  ]                     │
│          ◯            (OFF: Gray / ON: Green)           │
│        0.0 dB                                           │
└──────────────────────────────────────────────────────────┘
```

### カラースキーム

- **背景**: ダークグレー (#3C3C3C)
- **グループボックス**: ミディアムグレー (#505050)
- **タイトルラベル**: ライトグレー (#B0B0B0)
- **値テキスト**: ホワイト (#FFFFFF)
- **スライダーハンドル**: ブルー (#2196F3)
- **Link ボタン ON**: グリーン (#4CAF50)
- **Link ボタン OFF**: グレー (#606060)

### コントロールタイプ

#### 1. スライダー（Pan パラメータ）

**外観**: 水平スライダー（200x15 ピクセル）

**操作方法**:
- **左クリック + ドラッグ**: 値を変更
- **Shift + ドラッグ**: 微調整モード（10倍精密）
- **ダブルクリック**: デフォルト値にリセット
  - Left Pan → -100 (L100)
  - Right Pan → +100 (R100)

**表示**:
- 両端に "L" / "R" ラベル
- スライダー上部に現在値表示: "L100", "C", "R50" など

#### 2. ノブ（Gain、Delay パラメータ）

**外観**: 回転ノブ（60x60 ピクセル）

**操作方法**:
- **左クリック + ドラッグ**: 円形に動かして値を変更
  - 上方向ドラッグ: 値を増やす
  - 下方向ドラッグ: 値を減らす
- **Shift + ドラッグ**: 微調整モード（10倍精密）
- **ダブルクリック**: デフォルト値にリセット
  - Gain → 0dB
  - Delay → 0ms

**表示**:
- ノブ上部に現在値表示
- Gain: "-∞ dB", "0.0 dB", "+3.0 dB"
- Delay: "0.0 ms", "10.0 ms", "100.0 ms"

#### 3. トグルボタン（Link L/R Gain）

**外観**: 長方形ボタン（80x25 ピクセル）

**操作方法**:
- **クリック**: ON/OFF 切り替え

**表示**:
- **OFF 状態**: グレー背景、ライトグレーテキスト "LINKED"
- **ON 状態**: グリーン背景、ホワイトテキスト "LINKED"

**機能**:
- ON: Left Gain と Right Gain が連動して変更される
- OFF: 独立して調整可能

## パラメータ詳細

### Left Pan / Right Pan

- **コントロール**: スライダー
- **範囲**: -100 (Full Left) 〜 +100 (Full Right)
- **デフォルト**:
  - Left Pan: -100
  - Right Pan: +100
- **単位**: なし（-100 〜 +100）

**説明**:
各入力チャンネルの定位を調整します。-100 で完全に左、0 でセンター（等パワー分配）、+100 で完全に右に定位します。

**使用例**:
- **ステレオ幅拡張**: Left = -100, Right = +100（デフォルト）
- **L/R スワップ**: Left = +100, Right = -100
- **モノラル化**: Left = 0, Right = 0
- **左寄り**: Left = -100, Right = 0

### Left Gain / Right Gain

- **コントロール**: ノブ
- **範囲**: -60dB (ミュート) 〜 +6dB
- **デフォルト**: 0dB (Unity Gain)
- **単位**: dB

**説明**:
各入力チャンネルの音量を調整します。-60dB で完全ミュート（-∞）、0dB で入力そのまま、+6dB で約2倍の音量になります。

**使用例**:
- **音量バランス調整**: Left = -3dB, Right = 0dB（左を少し小さく）
- **片チャンネルミュート**: Left = -60dB（左をミュート）
- **音量ブースト**: Left = +3dB, Right = +3dB（両方を少し大きく）

### Left Delay / Right Delay

- **コントロール**: ノブ
- **範囲**: 0ms 〜 100ms
- **デフォルト**: 0ms
- **単位**: ms（ミリ秒）

**説明**:
各入力チャンネルに遅延を加えます。ハース効果により、遅延したチャンネルの定位感が変化します。10〜30ms の遅延で自然な広がりが得られます。

**使用例**:
- **ハース効果**: Left = 0ms, Right = 10ms（右がわずかに遅延）
- **スラップバックエコー**: Left = 0ms, Right = 100ms（明確なエコー）
- **ステレオ幅調整**: Left = 5ms, Right = 5ms（両方を同じだけ遅延）

### Master Gain

- **コントロール**: ノブ
- **範囲**: -60dB (ミュート) 〜 +6dB
- **デフォルト**: 0dB
- **単位**: dB

**説明**:
最終出力の全体音量を調整します。Left/Right Gain の後に適用されます。

**計算式**:
```
最終出力 = (Left入力 × Left Gain + Right入力 × Right Gain) × Master Gain
```

**使用例**:
- **全体音量下げ**: Master = -6dB（両チャンネルともに半分の音量）
- **全体音量上げ**: Master = +3dB（両チャンネルともに1.4倍の音量）

### Link L/R Gain

- **コントロール**: トグルボタン
- **値**: OFF / ON
- **デフォルト**: OFF

**説明**:
Left Gain と Right Gain を連動させます。

**動作**:
- **OFF**: Left/Right Gain を独立して調整可能
- **ON**:
  - Left Gain を変更 → Right Gain も同じ値に変更
  - Right Gain を変更 → Left Gain も同じ値に変更

**使用例**:
- ステレオバランスを保ったまま全体の音量を調整したい場合に ON
- 左右で異なる音量設定をしたい場合は OFF

## 使用例

### 例1: ワイドステレオイメージの作成

**目的**: ステレオ感を強調し、広がりのある音像を作る

**設定**:
```
Left Pan    = -100 (Full Left)
Right Pan   = +100 (Full Right)
Left Gain   = 0dB
Right Gain  = 0dB
Left Delay  = 10ms
Right Delay = 0ms
Master Gain = 0dB
```

**効果**: 左チャンネルが10ms遅延することで、ハース効果により音像が広がります。

---

### 例2: L/R チャンネルのスワップ

**目的**: ステレオ録音の左右を入れ替える

**設定**:
```
Left Pan    = +100 (Full Right)
Right Pan   = -100 (Full Left)
Left Gain   = 0dB
Right Gain  = 0dB
Left Delay  = 0ms
Right Delay = 0ms
Master Gain = 0dB
```

**効果**: 左入力が右から、右入力が左から聞こえます。

---

### 例3: 左チャンネルを強調

**目的**: 左チャンネルの音を大きくし、右を小さくする

**設定**:
```
Left Pan    = -100 (Full Left)
Right Pan   = +100 (Full Right)
Left Gain   = +3dB
Right Gain  = -6dB
Left Delay  = 0ms
Right Delay = 0ms
Master Gain = 0dB
```

**効果**: 左チャンネルが約1.4倍、右チャンネルが約0.5倍の音量になります。

---

### 例4: センター定位のモノラル化

**目的**: ステレオ信号をセンターでモノラルにまとめる

**設定**:
```
Left Pan    = 0 (Center)
Right Pan   = 0 (Center)
Left Gain   = 0dB
Right Gain  = 0dB
Left Delay  = 0ms
Right Delay = 0ms
Master Gain = 0dB
```

**効果**: 左右の入力がセンターに定位し、モノラルライクな音像になります。

---

### 例5: スラップバックエコー効果

**目的**: 片チャンネルに明確なエコーを加える

**設定**:
```
Left Pan    = -100 (Full Left)
Right Pan   = +100 (Full Right)
Left Gain   = 0dB
Right Gain  = -3dB
Left Delay  = 0ms
Right Delay = 80ms
Master Gain = 0dB
```

**効果**: 右チャンネルが80ms遅延し、明確なエコーが聞こえます。

## トラブルシューティング

### 問題: プラグインが DAW に表示されない

**原因**:
- プラグインファイルが正しいフォルダにない
- DAW がプラグインをスキャンしていない

**解決策**:
1. プラグインファイルが正しい VST3 フォルダにあることを確認
2. DAW のプラグインスキャンを実行
3. DAW を再起動

---

### 問題: GUI が開かない、またはクラッシュする

**原因**:
- グラフィックスドライバの問題
- DAW との互換性問題

**解決策**:
1. グラフィックスドライバを最新版に更新
2. DAW を最新版に更新
3. 別の DAW で試してみる

---

### 問題: 音が出ない

**チェックリスト**:
- [ ] トラックが有効になっているか
- [ ] Master Gain が -60dB (ミュート) になっていないか
- [ ] Left/Right Gain が -60dB になっていないか
- [ ] 入力信号が正しく送られているか

**解決策**:
1. すべての Gain を 0dB に設定
2. Pan をデフォルト値にリセット（Left=-100, Right=+100）
3. Delay を 0ms に設定

---

### 問題: 音にクリックノイズ/ポップノイズが入る

**原因**:
- パラメータの急激な変更
- バッファサイズが小さすぎる

**解決策**:
1. パラメータをゆっくり変更する
2. Shift + ドラッグで微調整モードを使う
3. DAW のバッファサイズを大きくする（512 samples 以上推奨）

---

### 問題: オートメーションが正しく動作しない

**確認事項**:
1. DAW がオートメーションを正しく記録しているか
2. プラグインがオートメーション対応パラメータを公開しているか

**解決策**:
1. DAW のオートメーション設定を確認
2. プラグインを再挿入
3. DAW のマニュアルでオートメーション方法を確認

---

### 問題: L/R Gain Link が効かない

**確認事項**:
1. Link ボタンが ON (グリーン) になっているか
2. Link ON の後に Gain を変更したか

**解決策**:
1. Link ボタンをクリックして ON にする（グリーン表示）
2. Left Gain または Right Gain を変更
3. もう一方の Gain も同じ値に変更されることを確認

## 付録

### キーボードショートカット

SimplePanner は以下のマウス操作に対応しています：

- **Shift + ドラッグ**: 微調整モード（10倍精密）
- **ダブルクリック**: デフォルト値にリセット

### パラメータ一覧表

| パラメータ | タイプ | 範囲 | デフォルト | 単位 |
|-----------|--------|------|-----------|------|
| Left Pan | スライダー | -100 〜 +100 | -100 | - |

- **サンプルレート対応**: 22.05kHz 〜 384kHz 以上

- **レイテンシー**: 遅延パラメータの最大値を報告
- **プロセッシング**: ステレオ入力 → ステレオ出力
